# Jupyter Notebooks

Interactive tutorials for learning optimal execution strategies.

## Available Notebooks

1. **01_introduction.ipynb** - Getting started with optimal execution
2. **02_calibration.ipynb** - Parameter calibration tutorial
3. **03_solver_comparison.ipynb** - Algorithm comparison
4. **04_monte_carlo.ipynb** - Statistical validation
5. **05_case_studies.ipynb** - Real-world examples

## Usage

```bash
# Install Jupyter
pip install jupyter

# Launch notebook server
jupyter notebook notebooks/

# Or use VS Code with Jupyter extension
```

## Requirements

All notebooks require the main package installed:
```bash
pip install -e .
```
